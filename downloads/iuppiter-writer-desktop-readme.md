# Iuppiter Writer — Desktop Version (Beta)

A native .docx document editor built with Python and tkinter.
Opens and saves real Microsoft Word (.docx) files on your desktop.

---

## Requirements

- **Python 3.8+** (https://www.python.org/downloads/)
- **python-docx** library
- **Pillow** library (for image support)

## Installation

1. Install Python 3.8 or newer from https://www.python.org/downloads/
   - On Windows: check "Add Python to PATH" during installation

2. Install dependencies:
   ```
   pip install python-docx Pillow
   ```

3. Run the editor:
   ```
   python iuppiter_writer.py
   ```

---

## Features

- Open and save real .docx files (Microsoft Word compatible)
- Rich text formatting: bold, italic, underline, strikethrough
- Font family and size selection
- Text color and highlight
- Paragraph styles: Normal, Heading 1–6, Title, Subtitle, Quote
- Paragraph alignment: left, center, right, justified
- Bullet and numbered lists
- Undo / Redo
- Find & Replace
- Dark mode toggle
- Print support
- Recent files list
- Page view with rulers

---

## Beta Disclaimer

This is BETA software — an early release.

**Not yet supported:**
- Track Changes, Comments, Review features
- Mail Merge
- Macros
- Some complex .docx formatting may not render perfectly

Use at your own risk. Always keep backups of important documents.
This software is not a replacement for Microsoft Word for mission-critical work.
