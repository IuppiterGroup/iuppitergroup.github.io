Installation

Five steps. Takes about two minutes on any machine with Python installed.

  1. Install Python 3.8+

    If you don't already have Python, download it free from python.org. Make sure to check "Add Python to PATH" during installation on Windows.

  2. Extract the .zip

    Right-click iuppiter-writer-desktop.zip → Extract All (Windows) or double-click (Mac). Place the extracted folder wherever you like.

  3. Install dependencies

    Open a terminal / command prompt and run:
    pip install python-docx Pillow
    This installs the two required libraries (one-time, ~5 MB).

  4. Launch the app

    Windows: Double-click luppiterWriter.bat
    Mac / Linux: In terminal, run python iuppiter_writer.py

  5. Open a document

    Click File → Open to load an existing .docx, or start a new blank document. Save back to .docx anytime with Ctrl+S.
