Hush

Cookie banners, GDPR popups, newsletter nags — silenced automatically on every site you visit.

Hush detects and dismisses the endless parade of consent dialogs, cookie banners, and overlay nags that modern websites throw at you — automatically, before you ever see them. It recognizes all major consent management platforms plus hundreds of generic patterns, clicks "Accept" on your behalf, or hides banners that don't respond to clicks.


A per-site whitelist lets you preserve popups on sites where you want them. The live counter shows how many popups Hush has eliminated today and all-time. Everything runs in your browser. Nothing is sent anywhere.

  • Blocks 50+ consent frameworks — OneTrust, CookieBot, Quantcast, TrustArc, Didomi, Osano, Complianz, iubenda, Klaro, and more
  • Auto-clicks "Accept" or hides the banner when no button is found
  • Supports English, German, French, Spanish, and Italian consent text
  • Per-site whitelist — keep popups on sites you trust
  • Live counter: blocked today & all-time
  • One-click pause / resume from the toolbar
  • MutationObserver — catches dynamically loaded banners
  • Zero data collection · Zero analytics · Fully offline

Tab Manager

Real-time memory usage for every open tab. Find the hogs, suspend the sluggards, reclaim your RAM.

Modern browsers are memory hogs. Iuppiter Tab Manager gives you full visibility into which tabs are responsible, sorted by memory usage in real time. When Chrome is consuming 6 GB of RAM and your system is starting to slow, you want to know which three tabs are responsible — and close them in one click.


Suspend individual tabs or bulk-close everything above a threshold you set. Group tabs by domain to see which sites are cumulatively draining your resources. An alert badge appears on the toolbar icon when total memory exceeds your limit. Optional auto-suspend runs silently in the background.

  • Real-time memory usage per tab — MB or GB, color-coded by severity
  • Sort all open tabs by memory, highest first
  • Suspend individual tabs instantly to free RAM
  • Bulk close: kill every tab over X MB with one click
  • Group view — see total memory consumption by domain
  • Search / filter across all open tabs by title or domain
  • Configurable alert threshold — toolbar badge turns red when exceeded
  • Optional auto-suspend for background tabs over your limit
  • Adjustable refresh interval (5–300 seconds)
  • No data transmitted anywhere · All processing is local

Mail Tracker

WhatsApp-style ✓✓ status indicators for Gmail. Know the moment your email gets a reply — no server required.

Ever sent an email and had no idea if it was ever read? Iuppiter Mail Tracker brings WhatsApp-style ✓ / ✓✓ status indicators directly into Gmail's sent view. It polls your sent threads via the Gmail API and updates each email's status automatically — no server, no subscription, no third-party service.


The popup dashboard shows your full tracking history: total tracked, reply rate, and per-email status. Filter by All / Replied / Pending. Desktop notifications fire the moment a reply comes in. You can pause tracking on any email individually, or export all your data as JSON. Everything runs in your browser and Chrome's local storage.

Note: This extension requires a free Gmail API OAuth credential (Client ID) to function. Setup takes ~5 minutes — see the included README.md for step-by-step instructions. No data is shared with us or any third party.

  • ✓ / ✓✓ status badges injected directly into Gmail's sent view
  • Reply detection — gold ✓✓ when recipient replies to your thread
  • Read receipt detection — blue ✓✓ when Gmail read receipt is confirmed
  • Dashboard popup with stats: total tracked, replied, read count, reply rate %
  • Filter by status — All / Replied / Pending / Read Receipt
  • Desktop notifications the moment a reply arrives
  • Per-email toggle — pause tracking for individual sent messages
  • Export all tracking data as JSON
  • Auto-polls every 5 minutes in the background
  • 100% local — all data in chrome.storage.local, no external servers
  • Gmail read-only OAuth scope — never modifies your email

Bookmarks Manager

Visual bookmark management with tags, collections, dead link detection, and full-text search. Your links, finally organized.

Tags, notes, and collections so you can actually organize what you've saved. Auto-categorization assigns every new bookmark to a category (news, dev, finance, social, video, and more) based on its URL and title.

Full-text search covers titles, URLs, tags, and your personal notes. Duplicate detection warns you before saving a URL you already have. Clipboard Manager

Persistent clipboard history with smart categorization, pinning, and instant search. Never lose a copied snippet again.

Every time you copy text, Iuppiter Clipboard captures it and keeps it organized in a searchable history. URLs, emails, phone numbers, code snippets, and plain text are automatically categorized so you can filter by type in one click. The popup opens instantly with Ctrl+Shift+C, and clicking any entry pastes it back to your clipboard immediately.


Pin critical snippets to keep them at the top regardless of age — connection strings, API endpoints, template text you use constantly. Auto-clear settings let you keep your history lean, and full export means your data is always yours. Privacy-first by design: password fields are never captured, full stop.

  • Captures copy events from every page — automatically and silently
  • Smart auto-categorization: URL, Email, Phone, Code, Text
  • Pin items to keep them permanently at the top of your history
  • Full-text search across your entire clipboard history
  • Expand / collapse long entries without leaving the popup
  • Filter tabs by category — All, Text, URLs, Emails, Phone, Code
  • Export full history as a .txt file for archiving or migration
  • Configurable max history size (10–1000 items)
  • Auto-clear stale items after 1 / 3 / 7 / 14 / 30 days, or never
  • Keyboard shortcut: Ctrl+Shift+C (Mac: Cmd+Shift+C)
  • Password fields are never captured — always enforced, not configurable
  • 100% local · No server · No analytics · chrome.storage.local only

Privacy & Terms

🔒 Privacy Policy

Applies to all Iuppiter Group Chrome extensions.

  • No personal data collected. Extensions do not access, read, or store any personally identifiable information.
  • No analytics or tracking. We use no analytics libraries, no telemetry, no crash reporting services.
  • All processing is local. Everything runs inside your browser. No data is transmitted to any server — including ours.
  • Storage stays on your device. Preferences (whitelist entries, settings) are saved in Chrome's local storage and never leave your machine.
  • No third-party SDKs. Extensions contain no embedded third-party code that could collect data.
  • Permissions explained: Extensions request only the permissions required for their stated function. See each extension's manifest for the complete list.
  • Mail Tracker exception: Uses Gmail read-only OAuth to access your own sent mail. No email content is stored, transmitted, or shared. OAuth tokens are managed by Chrome and never leave your browser.
  • Dashboard widget data: Weather and stock widgets fetch from public APIs (wttr.in, Yahoo Finance) on your behalf. No identifying information is sent — only the city name or ticker symbols you configure.
