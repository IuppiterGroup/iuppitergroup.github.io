Features

Two versions, one vision: a clean, capable writing environment

Web Version (Free)

  • Runs entirely in the browser — no install
  • Microsoft Word-style ribbon interface
  • Bold, Italic, Underline, Strikethrough
  • Font family and size picker
  • Text color and highlight
  • Paragraph alignment (L / C / R / Justify)
  • Bullet and numbered lists
  • Heading styles (H1–H6)
  • Find & Replace
  • Table insertion
  • Image insertion
  • Print / Export to PDF via browser
  • Open and save .docx files (via JSZip)
  • Page layout view with ruler
  • Header & Footer editing
  • Word and character count
  • Keyboard shortcuts (Ctrl+B, S, Z, Y, etc.)
  • Dark mode
  • Works offline after download

Desktop Version ($5)

  • Native .docx read/write — true Word compatibility
  • Full round-trip fidelity (open → edit → save)
  • python-docx powered — industry-standard library
  • Rich text formatting with per-character precision
  • Paragraph styles: Normal, Heading 1–6, Title, Subtitle, Quote
  • Line spacing control
  • Indent / outdent
  • Bullet and numbered lists
  • Font family and size
  • Text color and highlight
  • Tables (open existing)
  • Image rendering (via Pillow)
  • Undo / Redo stack
  • Find & Replace
  • Print support
  • Recent files list
  • Dark mode toggle
  • Runs on Windows, Mac, Linux
