Getting Started

  1. Download the HTML file
  2. Open it in any modern browser (Chrome, Firefox, Edge, Safari)
  3. Enter a ticker symbol and current financial metrics (EPS or FCF per share)
  4. Adjust growth rate, WACC, terminal multiple, and buyback assumptions
  5. View Bear / Base / Bull scenario valuations with full projection tables
  6. Run Monte Carlo simulations for probability-weighted ranges
  7. Export results as CSV for further analysis

Features

  1. Three-scenario modeling — Bear, Base, and Bull with customizable offsets
  2. Monte Carlo simulation — 10,000 randomized runs for probability distributions
  3. EPS or FCF basis — Toggle between earnings-per-share and free-cash-flow
  4. Share buyback modeling — Factor in annual share repurchase rates
  5. Terminal value calculation — Exit multiple approach with adjustable parameters
  6. CSV export — One-click export of all projections and scenarios
  7. Fully offline — No internet required after download, no data sent anywhere
